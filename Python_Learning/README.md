# Python Learning

Python fundamentals learning project — covers syntax, data types, operators, string methods, collections, naming conventions, and API testing. Designed for QA/test automation engineers.

## Files

| File | Topic | Description |
|------|-------|-------------|
| `hello_world.py` | Print statement | Classic first program |
| `Basic.py` | Variables & Types | Dynamic typing, `type()`, f-strings, basic data types |
| `Basic2.py` | Type Casting | `int()`, `str()`, boolean conversion |
| `Identifier.py` | Naming Rules | 10 identifier rules & PEP 8 conventions |
| `Keyword.py` | Reserved Keywords | Lists all 35 Python keywords |
| `Literals.py` | Literal Types | Numeric, string, boolean, None, collections, special literals |
| `Operators.py` | Operators | Arithmetic, comparison, logical, identity, membership, ternary, bitwise, precedence |
| `Strings.py` | String Methods | Case, trim, search, split/join, formatting |
| `Tuple.py` | Tuple vs List | Immutable vs mutable collections |
| `List_Tuple_Dict.py` | Collections | Placeholder for list/tuple/dict examples |
| `Case.py` | Naming Conventions | snake_case, PascalCase, camelCase, and more |
| `AIAGent_Query.py` | AI-themed Demo | Model names, f-string prompts, LLM config references |
| `test_api.py` | API Testing | Real HTTP test against JSONPlaceholder with `requests` |

## Prerequisites

- Python 3.11+
- `requests` library

## Setup

```bash
python -m venv venv
venv\Scripts\activate
pip install requests
```

## Run

```bash
venv\Scripts\activate
python hello_world.py
python test_api.py
```
